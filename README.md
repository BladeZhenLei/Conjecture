# Conjectures

## 1. Prime Tower
Are all numbers of the form 
$$2^{2^{2^{2^{{\cdot}^{{\cdot}^{\cdot}}}}}}...+3^{3^{3^{3^{{\cdot}^{{\cdot}^{\cdot}}}}}}$$ 
primes? 
<p/> 
Since 
$$p_1=2+3=5,$$ 
$$p_2=2^{2}+3^{3}=31,$$ 
$$p_3={2^2}^{2}+{3^3}^{3}=7625597485003,$$
are all primes.
<p/>
(Note that the number of digits for this number is approximately ${10^10}^12.56$, it wil be impossible to calculate the number itself, let along do primality test!)
